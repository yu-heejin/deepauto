# Multi Agent Workflow System 구현

## 사용 기술

- FastAPI + Python (3.11)
- MySQL
- Docker
- docker-compose

## 실행 방법

```json
docker-compose up -d
```

## API 실행 결과

`POST /workflows` 

### Response

    "status": 201,
    "data": {
